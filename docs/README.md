# NewsHub

一个聚合多个新闻网站热点新闻的现代化平台。

## 项目简介

NewsHub 致力于为用户提供一站式的新闻阅读体验，通过聚合多个主流新闻网站的热点内容，帮助用户快速了解最新的资讯动态。

## 核心特性

- **多源聚合** - 整合多个新闻网站的内容
- **实时热点** - 智能算法追踪和推荐热点新闻
- **分类浏览** - 支持按类别（科技、财经、娱乐等）浏览
- **搜索功能** - 快速查找感兴趣的新闻
- **响应式设计** - 适配多种设备和屏幕尺寸
